# PopupPilot Documentation

## Setup
1. Upload the plugin to `/wp-content/plugins/`.
2. Activate it via the Plugins menu.
3. Go to PopupPilot > Settings to configure your OpenAI API key for AI features.

## Creating Popups
- Use the visual editor to drag and drop components.
- Configure targeting rules (Device, URL, Geo, UTM) in the side panel.
- Use the AI Assistant to generate or modify layouts via natural language.

## Integrations
- Configure Mailchimp, HubSpot, or ConvertKit in the Integrations tab.
- Form submissions will automatically sync to your selected provider.
- Use Webhooks for custom integrations.

## Analytics
- Track impressions, views, clicks, and conversions in real-time.
- Export your data to CSV for further analysis.
